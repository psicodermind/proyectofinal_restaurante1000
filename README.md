## PlatoClick - Proyecto Final Geraldine Ramirez 
repositorio: https://github.com/psicodermind/proyectofinal_restaurante1000.git

Aplicación web para explorar restaurantes, sus platos, pedidos y clientes.
Desarrollada con React + Vite y desplegada en GitHub Pages.

🌐 Ver en Producción
Accede directamente desde el navegador:
👉 https://psicodermind.github.io/proyectofinal_restaurante1000/ 
No necesitas instalar nada, siempre y cuando el servido de la base de datos este funcionando. 

💻 Ejecutar en Local
Requisitos previos

Node.js instalado o actualizado 
Docker instalado o actualizado 

Pasos
1. Levantar el Docker
docker compose up -d

2. Ejecutar en modo desarrollo
npm run dev 


Desplegar en Producción
Desde la carpeta restaurante-frontend/:
npm run deploy


Tecnologías

React 19
Vite 7
React Router DOM
React Bootstrap
GitHub Pages (gh-pages)
